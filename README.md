I'm making each of my plugins its own repository instead of all of them being condensed into one for 'ease of use'. That simply made everything harder to keep track of.

This repository is a balance mod for TF2. The link to the document upon which this work is based will be included in here once said document is in a state of relative completion. This may take forever depending on how motivated I feel.

Dependencies for this plugin:
* tf2items
* tf2utils
* tf2attributes

I'll try to get rid of TF2Items as a dependency at some point. TF2Utils has other dependencies too. Make sure everything is installed.
